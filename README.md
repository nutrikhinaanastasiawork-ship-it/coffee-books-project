# Анализ продаж магазина "Кофе и Книги"

## Студенты
- **Студент А** — товарный анализ (категории, товары, цены, ABC-анализ)
- **Студент Б** — временной анализ (дни недели, будни/выходные, тренды)

## Структура проекта
- `data/` — исходные данные
- `scripts/student_A/` — скрипты товарного анализа
- `scripts/student_B/` — скрипты временного анализа
- `outputs/` — результаты (графики, метрики)
- `final_report.Rmd` — итоговый отчёт

## Как запустить
1. Открыть `scripts/00_setup.R` и выполнить
2. Запустить `scripts/01_data_prep.R`
3. Запустить `scripts/student_A/A4_run_all.R` и `scripts/student_B/B4_run_all.R`
4. Скомпилировать `final_report.Rmd`
